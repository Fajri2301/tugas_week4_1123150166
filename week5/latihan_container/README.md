# Latihan Container Flutter

Proyek Flutter untuk latihan membuat layouting menggunakan container dan widget lainnya.

## Langkah-langkah Pengembangan

### 1. Struktur Awal dan Splash Screen
- **File Utama (`lib/main.dart`):** Mengatur routing aplikasi dan tema global. Halaman awal diatur ke `SplashScreen`.
- **Splash Screen (`lib/splash_screen.dart`):** Halaman pertama yang dilihat pengguna. Berisi gambar dan tombol untuk melanjutkan ke halaman utama.
